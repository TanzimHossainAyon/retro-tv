# 📺 Retro CRT TV Simulator

A fully interactive retro CRT television simulator with 7 channels, remote control, and authentic scanline effects!

## 👤 Author
**Tanzim Hossain Ayon** | CSE Graduate · BRAC University · IEEE Researcher
🌐 [Portfolio](https://tanzimhossainayon.vercel.app)

## 🚀 Features
- 📺 Realistic CRT TV design with scanlines & phosphor glow
- 🎮 Virtual remote control with number pad
- 📡 7 channels — About Me, Skills, Research, Weather, Music, Contact, No Signal
- ⚡ Power on/off animation
- 📻 Music visualizer on CH 5
- 🌦️ Live weather display on CH 4
- ⌨️ Keyboard shortcuts (Arrow keys, numbers, P, M)
- 💾 Last channel saved via LocalStorage
- 📊 Noise canvas animation

## ⌨️ Keyboard Shortcuts
- `↑↓` — Change channel
- `←→` — Volume
- `0-9` — Go to channel
- `P` — Power on/off
- `M` — Mute

## 🛠️ Technologies
HTML5 · CSS3 · JavaScript · Canvas API · LocalStorage · Web Animations · CSS Variables · VT323 Font

## ▶️ How to Run
Open `index.html` in any browser!
